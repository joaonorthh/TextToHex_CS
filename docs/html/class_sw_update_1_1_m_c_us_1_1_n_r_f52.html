Operational codes of the NRF52 DFU Protocol:

  • 0x00 - Protocol Version
  • 0x01 - CreateObject
  • 0x02 - SetPRN
  • 0x03 - CalcChecSum
  • 0x04 - Execute
  • 0x05 - ReadError
  • 0x06 - Select/ReadObject
  • 0x07 - GetSerialMTU
  • 0x08 - WriteObject
  • 0x09 - Ping
  • 0x60 - Response
  • 0xC0 - End
