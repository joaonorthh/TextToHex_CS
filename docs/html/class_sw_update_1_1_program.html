◆ Main()

static void SwUpdate.Program.Main ( string[]  args)
staticprivate

p01-update -p PATH, updates the Sensor Box / Connectector Box automatically with the files in that folder. p01-update -sw WCS -p PATH, where PATH is an optional parameters to the FOLDER with the WCS-file in it p01-update -sw ALL -p PATH p01-update -sw UIS -p PATH p01-update -sw CBS -p PATH p01-update -sw SBS -p PATH This commandline tool updates the medical device smart-sync.

Get the COM PORT:

Types: 0=SB; 1=CB; 11=cannot open port.
